# Venus Engine — Contexto do Produto

Venus Engine é um sistema premium de análise de imagem, estilo, paleta, biotipo, visagismo, acessórios e recomendação de looks.

Tem duas frentes:
- B2C: experiência premium para o cliente final
- B2B: sistema para lojistas venderem mais com recomendações inteligentes

Objetivos do MVP:
- UX premium
- mobile-first
- clareza absoluta
- redução de fricção
- foco em decisão de compra
- base pronta para integração futura com InovaCortex

Princípios:
- não complicar a arquitetura
- separar B2C e B2B
- priorizar componentes reutilizáveis
- sempre pensar em conversão, não só em estética
