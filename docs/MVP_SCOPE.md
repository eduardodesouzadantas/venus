# Venus Engine — MVP Scope

## Objetivo do MVP
Lançar uma primeira versão premium, funcional e vendável do Venus Engine com foco em experiência B2C excepcional e base B2B simples.

## O que entra no MVP

### B2C
- onboarding mobile-first
- captura de foto de rosto
- captura de foto de corpo
- análise visual complementar
- result dashboard
- 3 looks gerados
- acessórios essenciais e premium
- CTA para vitrine
- opção de salvar resultado / criar conta

### B2B
- login simples do lojista
- cadastro manual de produtos
- catálogo com tags básicas
- painel simples com produtos disponíveis para recomendação

### Lógica
- look generation logic
- hybrid recommendation engine simplificado
- accessory engine simplificado

## O que NÃO entra no MVP
- closet completo automatizado
- múltiplas variações por look
- analytics avançado
- campanhas automáticas
- integração completa com WhatsApp
- integração com ERP/estoque externo
- checkout complexo
- IA tentando analisar psicologia por foto

## Regra de ouro
O MVP deve ser bonito, confiável, rápido e vendável.
Não deve tentar resolver tudo na primeira versão.
