# Venus Engine — Build Order

## Ordem obrigatória de construção

### Fase 1 — fundação
1. criar estrutura do projeto
2. configurar design system base
3. criar layout mobile-first
4. criar rotas principais

### Fase 2 — onboarding
5. criar telas do onboarding
6. persistir respostas
7. criar fluxo entre etapas

### Fase 3 — scanner
8. criar telas de captura
9. criar upload de fotos
10. criar fluxo de processamento visual

### Fase 4 — inteligência
11. implementar look generation logic
12. implementar hybrid recommendation engine simplificado
13. implementar accessory engine simplificado

### Fase 5 — resultado
14. criar result dashboard
15. renderizar paleta, diagnóstico e looks
16. adicionar CTA final

### Fase 6 — b2b
17. criar login simples do lojista
18. criar cadastro manual de produtos
19. criar catálogo interno utilizável pela IA

## Regra de execução
Nunca construir tudo de uma vez.
Sempre finalizar uma fase antes de avançar.
